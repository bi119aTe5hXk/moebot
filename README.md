##注意事项
1.目前Program-O对Unicode的兼容性暂时不佳，所以*和_的wildcard字符可能会无法正常识别，编写时建议减少该用法或换用其他AIML引擎（至少目前Program-O并不好用）。

2.由于文件中中可能会含有大量系统级操作相关命令。所以请仅在自家本地服务器或设备上运行。如果将文件应用于公开的bot，有导致被恶意攻击的可能。
